# Tuần 4: Quản lý vận hành và hạ tầng dưới dạng mã

**Thời gian:** 08/05/2026 - 14/05/2026

## 1. Mục tiêu trong tuần

Tuần 4 tập trung vào việc quản lý tài nguyên AWS, tự động hóa vận hành hệ thống và triển khai hạ tầng bằng code.

Các nội dung chính trong tuần gồm: AWS Toolkit for VS Code, Tags, Resource Groups, IAM Resource Tags, AWS Systems Manager, Session Manager, AWS Lambda, AWS CloudFormation và AWS CDK.

## 2. Nội dung thực hiện

### Môi trường phát triển với AWS Toolkit for VS Code

Tìm hiểu cách sử dụng AWS Toolkit trong Visual Studio Code để hỗ trợ làm việc với các dịch vụ AWS. Công cụ này giúp thao tác với tài nguyên Cloud thuận tiện hơn ngay trong môi trường lập trình.

### Tổ chức tài nguyên với Tags và Resource Groups

Tìm hiểu cách gắn tag cho tài nguyên AWS để phân loại, quản lý và theo dõi chi phí dễ hơn. Resource Groups giúp gom các tài nguyên có cùng đặc điểm để quản lý tập trung.

### Kiểm soát quyền truy cập với IAM và Resource Tags

Tìm hiểu cách kết hợp IAM với tag để kiểm soát quyền truy cập tài nguyên. Nội dung này giúp hiểu rõ hơn cách giới hạn quyền theo nhóm tài nguyên, môi trường hoặc mục đích sử dụng.

### Quản lý hệ thống với AWS Systems Manager

Tìm hiểu AWS Systems Manager và vai trò của dịch vụ này trong quản lý máy chủ, cấu hình hệ thống và tự động hóa tác vụ vận hành. Đây là công cụ hữu ích để quản lý nhiều tài nguyên AWS tập trung.

### Truy cập máy chủ với Session Manager

Tìm hiểu Systems Manager Session Manager để truy cập máy chủ EC2 mà không cần mở cổng SSH trực tiếp. Cách này giúp tăng bảo mật và giảm rủi ro khi quản trị máy chủ.

### Tự động hóa bằng AWS Lambda

Tìm hiểu AWS Lambda và mô hình serverless. Nội dung này giúp nắm được cách chạy code mà không cần quản lý máy chủ, phù hợp cho các tác vụ tự động hóa hoặc xử lý sự kiện.

### Hạ tầng dưới dạng mã với AWS CloudFormation

Tìm hiểu AWS CloudFormation để tạo và quản lý tài nguyên AWS bằng template. Nội dung này giúp giảm thao tác thủ công và đảm bảo hạ tầng được triển khai nhất quán.

### Bộ công cụ phát triển đám mây với AWS CDK

Tìm hiểu AWS CDK ở mức cơ bản và nâng cao. CDK cho phép định nghĩa hạ tầng bằng ngôn ngữ lập trình, giúp việc xây dựng và tái sử dụng kiến trúc Cloud linh hoạt hơn.

### Workshop Infrastructure as Code

Thực hành tư duy Infrastructure as Code thông qua việc mô tả, triển khai và quản lý tài nguyên bằng code. Nội dung này giúp hiểu rõ hơn lợi ích của tự động hóa hạ tầng trong môi trường thực tế.

## 3. Kết quả đạt được

Sau tuần 4, đã nắm được cách quản lý tài nguyên AWS hiệu quả hơn thông qua tag, resource group và các công cụ vận hành.

Ngoài ra, nội dung về CloudFormation và AWS CDK giúp hiểu được cách triển khai hạ tầng bằng code, giảm lỗi cấu hình thủ công và hỗ trợ tái sử dụng trong các dự án sau.

## 4. Nhận xét

Tuần 4 cung cấp kiến thức quan trọng về quản lý vận hành và tự động hóa hạ tầng. Các nội dung như Systems Manager, Lambda, CloudFormation và CDK là nền tảng cần thiết để xây dựng hệ thống Cloud có khả năng quản lý tốt, dễ mở rộng và dễ bảo trì.
